Los Elementos Flotantes

Float

Es una etiqueta de CSS que permite ubicar y mover un elemento en torno a un texto. Interviene en el flujo normal del texto y se integra en él según el valor que se le dé. Este elemento se llama elemento flotante y dependiendo del valor otorgado modifica el display.

  • Block: inline, inline-block, inline-table, inline-row, inline-row-group, table-column, table-column-group, table-cell, table-caption, table-header-group, table-footer-group.
  • Flex: inline-flex.
  • Grid: inline-grid.

Su sintaxis es sencilla, va en la parte de CSS, tanto en el archivo Style, como dentro de la etiqueta.

Sería tal que así:

Aaaaa{ float: right; }

    Sus valores posibles son:
  • Left: El elemento debe flotar a la izquierda de su bloque contenedor.
  • Right: El elemento debe flotar a la derecha de su bloque contenedor.
  • Inline-start: El elemento debe flotar en el costado de inicio de su bloque contenedor.
  • Inline-end: El elemento debe flotar en el costado de término de su bloque contenedor.
  • None: El elemento no deberá flotar. Siendo este último el default.

Clear

Es una etiqueta de CSS que especifica si un elemento puede estar al lado de elementos flotantes o debe ser movido debajo de ellos.

Sería tal que así:

Aaaaa{ clear: right; }

    Sus valores posibles son:
  • Right: El elemento es movido hacia abajo para limpiar elementos flotantes a la derecha.
  • Left: El elemento es movido hacia abajo para limpiar elementos flotantes a la izquierda.
  • Both: el elemento es movido hacia abajo para limpiar tanto elementos flotantes de la izquierda como de la derecha.
  • Inline-start: El elemento es movido hacia abajo para limpiar elementos flotantes al inicio del bloque contenedor.
  • Inline-end: El elemento es movido hacia abajo para limpiar floats al final del bloque contenedor.
  • None: El elemento no es movido hacia abajo para limpiar elementos flotantes anteriores. Siendo este último el default.

Overflow

Es una etiqueta de CSS que especifica si recortar contenido, dibujar barras de desplazamiento o mostrar el contenido excedente en un elemento a nivel de bloque.

Su sintaxis es sencilla, va en la parte de CSS, tanto en el archivo Style, como dentro de la etiqueta.

Sería tal que así:

Aaaaa{ Overflow: visible; }

    Sus valores posibles son:
  • Visible: El contenido no es recortado.
  • Hidden: El contenido es recortado y no se muestran barras de posición.
  • Scroll: El contenido es recortado y el navegador web usa las barras de desplazamiento.
  • Auto: Depende del agente de usuario. Navegadores de escritorio como Firefox proveen barras de desplazamiento si hay contenido excedente.
