Etiquetas de HTML5

Las etiquetas HTML son elementos que contienen la información del documento. Se dividen en semánticas y básicas y se diferencian en que las primeras definen el contenido que engloban, en cambio las segundas solo encierran contenido.

SEMÁNTICAS

Metadatos del Documento:

  • <head>: Contenedor para metadatos.
  • <title>: Título de la página, visible en la pestaña del navegador.
  • <meta>: Metadatos sobre el documento (charset, viewport, etc.).
  • <link>: Enlace a recursos externos (CSS, favicon, etc.).
  • <style>: CSS dentro del HTML.

Estructura:

  • <header>: Encabezado de una sección o página.
  • <footer>: Pie de página de una sección o página.
  • <nav>: Contenedor de enlaces de navegación.
  • <main>: Contenido principal de la página.
  • <section>: Sección temática dentro de una página.
  • <article>: Contenido independiente, como una entrada de blog o un artículo.
  • <aside>: Contenido relacionado pero secundario, como una barra lateral.
  • <hgroup>: Agrupación de elementos de encabezado (<h1>, <h2>, etc.).

Texto y Media:

  • <p>: Párrafo. // <pre>: Texto preformateado.
  • <q>: Cita en línea. // <blockquote>: Cita en bloque.
  • <figcaption>: Pie de figura que acompaña a <figure>:para contenido ilustrativo.
  • <address>: Información de contacto.

Lista:

  • <ol>: Lista ordenada.
  • <ul>: Lista desordenada.
  • <li>: Elemento de lista.

Enlaces

  • <a>: Enlace a otra página o ubicación.

Tablas y Datos Tabulares:

  • <table>: Contenedor de tabla.
  • <caption>: Título de la tabla.
  • <thead>: Encabezado de la tabla.
  • <tbody>: Cuerpo de la tabla.
  • <tfoot>: Pie de la tabla.
  • <tr>: Fila de la tabla.
  • <th>: Encabezado de celda.
  • <td>: Celda de datos.

Formularios:

  • <form>: Formulario.
  • <input>: Entrada de datos.
  • <label>: Etiqueta para elementos de formulario.
  • <fieldset>: Agrupación de controles dentro de un formulario.
  • <legend>: Título de un grupo de elementos de formulario.
  • <select>: Lista desplegable.
  • <optgroup>: Agrupación de opciones en un desplegable.
  • <option>: Opción de un desplegable.
  • <textarea>: Área de texto.
  • <button>: Botón.

Multimedia:

  • <img>: Imagen. // <audio>: Audio. // <video>: Video.
  • <source>: Fuente de recursos multimedia (dentro de <audio> o <video>).
  • <track>: Texto para subtítulos, capítulos o descripciones para <video>.

Script y Programación:

  • <script>: Bloque de código JavaScript.
  • <canvas>: Área de dibujo para gráficos dinámicos en 2D y 3D.

BÁSICAS

Estructura:

  • <html>: Elemento raíz que envuelve todo el contenido de la página.
  • <body>: Contenedor del contenido principal visible en la página.

Encabezados:

  • <h1> a <h6>: Encabezados de diferentes niveles (Solo debe haber un <h1> por web).

Texto:

  • <b>: Negrita sin // <strong>: Negrita con (Énfasis semántico).
  • <i>: Cursiva sin // <em>: Cursiva con (Énfasis semántico).
  • <small>: Tamaño pequeño.
  • <mark>: Resaltado.
  • <br>: Salto de linea.
  • <del>: Tachado.
  • <ins>: Subrayado.
  • <hr>: Línea horizontal.

Contenido genérico:

  • <div>: División o contenedor para agrupar otros elementos.
  • <span>: Contenedor en línea para agrupar sin romper el flujo del texto.

Multimedia:

  • <embed>: Incrustación de contenido externo (por ejemplo, un archivo PDF).
  • <object>: Inserción de contenido multimedia o recursos externos.
  • <param>: Parámetros para elementos <object>.

Metadatos y Recursos Externos:

  • <base>: Establece la URL base para los enlaces relativos en el documento.

Etiquetas Obsoletas:

Algunas etiquetas que pueden encontrarse en sitios web antiguos, pero que están en desuso en HTML5:

  • <center>: Alineación centrada de contenido (reemplazada por CSS).
  • <font>: Personalización de fuentes (reemplazada por CSS).
  • <acronym>: Para definir acrónimos (reemplazada por <abbr>).
