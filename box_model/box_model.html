CSS Box Model

Consiste en una caja que envuelve cada elemento HTML e incluye: content, padding, borders and margins.

Explicación de las diferentes partes:

  • Content: el contenido del cuadro, donde aparecen el texto y las imágenes.
  • Padding: borra un área alrededor del contenido. El padding es transparente.
  • Border: borde que rodea el padding y el content.
  • Margin: borra un área fuera del borde. El margen es transparente.

El BOX MODEL permite agregar un borde alrededor de los elementos y definir el espacio entre elementos.

Ejemplo:

Este texto es el contenido de la caja. Tiene un padding de 4rem, un margin de 3rem y un borde azul de 2rem. El ancho de la caja es de 30rem y el alto es de 6rem. El color de fondo es rosa.
div {
 background-color: pink;
 width: 30rem;
 border: 2rem solid blue;
 padding: 4rem;
 margin: 3rem;
 height: 6rem;
}

CSS Outline

Outline es una línea que se dibuja alrededor de elementos FUERA de los bordes.

Ejemplo:

Propiedades:

  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

Outline es diferente a border. A diferencia de border la outline es dibujada fuera del borde del elemento y puede superponerse a otro contenido. Además outline no es parte de las dimensiones del elemento. El height y width del elemento no es afectado por el width del outline.

CSS Box Sizing

Esta propiedad permite incluir el padding y el border en el total del width y height de un elemento.

Por defecto, cuando no incluimos la propiedad Box Sizing a un elemento, se le suman el padding y el borde al ancho total del elemento:

width + padding + border = ancho real de un elemento

height + padding + border = alto real de un elemento

La propiedad Box Sizing permite agregar el border y el padding al width y height total del elemento
