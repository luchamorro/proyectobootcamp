CSS Grid Layout

El módulo de CSS Grid Layout ofrece un sistema basado en grillas con filas y columnas. Es una alternativa más sencilla a la utilización de floats y positioning.

Permite crear una grilla dividiendo un contenedor en diferentes áreas.

Display: un elemento HTML se convierte en un contenedor de rejilla (grid container) cuando la propiedad display se configura como grid o inline-grid.

Propiedades CSS de Grid Layout


  • column-gap - Especifica el gap entre columnas
  • gap - Shorthand para las propiedades row-gap y column-gap
  • grid - Shorthand para las propiedades grid-template-rows, grid-template-columns, grid-template-areas, grid-auto-rows, grid-auto-columns y grid-auto-flow.
  • grid-area - Se puede especificar los nombres de los elementos de la grilla o utilizar como shorthand para grid-row-start, grid-column-start, grid-row-end y grid-column-end.
  • grid-auto-columns - Especifica un tamaño de columna por defecto.
  • grid-auto-flow - Especifica como los elementos auto-ubicados se insertan en la grilla.
  • grid-auto-rows - Especifica un tamaño de fila por defecto.
  • grid-column - Shorthand para las propiedades grid-column-start y the grid-column-end.
  • grid-column-end - Especifica dónde termina el elemento de la grilla.
  • grid-column-gap - Tamaño del gap entre columnas.
  • grid-column-start - Comienzo del item de la grilla.
  • grid-gap - Shorthand para las propiedades grid-row-gap y grid-column-gap.
  • grid-row - Shorthand para las propiedades grid-row-start y grid-row-end.
  • grid-row-end - Especifica dónde termina el item rejilla.
  • grid-row-gap - Tamaño del gap entre filas.
  • grid-row-start - Especifica dónde empieza el item rejilla.
  • grid-template - Shorthand para las propiedades grid-template-rows, grid-template-columns y grid-areas.
  • grid-template-areas - Especifica cómo mostrar columnas y filas usando nombres de los elementos de la rejilla.
  • grid-template-columns - Especifica el tamaño de las columnas y la cantidad de columnas en una rejilla.
  • grid-template-rows - Especifica el tamaño de las filas en la rejilla.
  • row-gap - Especifica el gap entre las filas de la grilla.
